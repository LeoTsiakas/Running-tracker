# README

This a Ruby on Rails app where a user can enter his training metrics and track his progress over time.

# Docker

You can run this app in your local machine by following these simple steps

1. Clone the repo

```bash
git clone git@github.com:LeoTsiakas/Running-tracker.git
cd running-tracker
```

2. Build Docker image and run container

```bash
docker compose up
```

3. Visit app on http://localhost:3000
